1. Change Application properties 
spring.datasource.url = jdbc:mysql://34.72.134.220:3306/reportsdb
spring.datasource.username = digapp
spring.datasource.password = 2023
spring.jpa.hibernate.ddl-auto = update

spring.jpa.show-sql=true

2. Comment on test DemoApplicationTests.java @Test

3. Build the project to generate the JAR file, with maven use 
mvn clean
mvn package

You will see and JAR generated file on target foler, in the main project folder create the dockerfile
Dockerfile
FROM openjdk:8

COPY target/demo-0.0.1-SNAPSHOT.jar demo-0.0.1-SNAPSHOT.jar

ENTRYPOINT ["java", "-jar", "/demo-0.0.1-SNAPSHOT.jar"]

Create the image
docker build -t digapp_backend_reports_ms .

4. Start the springboot container on the same network


docker build -t digapp_backend_reports_ms .

sudo docker run --name digapp_backend_reports_ms -d -p 8081:8081 digapp_backend_reports_ms

7. Check if the deploynment was successfull

check container
docker ps
docker logs FISRT3IDCONTAINER

8. Do test on Postman
localhost:8081/api/reports
